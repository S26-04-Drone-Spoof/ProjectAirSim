# Project AirSim - Virginia Tech ECE Senior Design S26-04

This is a forked repository of [Project AirSim](https://github.com/iamaisim/ProjectAirSim) that has been modified to suit the needs of the **S26-04 Virginia Tech ECE Senior Design Team**.

## About This Fork

This repository contains customizations and modifications specific to our senior design project requirements. While maintaining the core functionality of Project AirSim, we have adapted the platform for our team's specific use cases and research objectives. Please see team website for more details.

## Original Project

Project AirSim is a simulation platform for drones, robots, and other autonomous systems built on Unreal Engine 5. It provides photo-realistic visuals and a comprehensive framework for developing autonomous systems.

For complete information about the original Project AirSim platform, please see [README_ORIGINAL.md](README_ORIGINAL.md).
